# cos

COS(Collection Of Suckless) is a tiny Linux distribution that aims to be as suckless as possible.

# Software

- Compiler: tcc
- Coreutils: sbase, ubase
- Shell: 90s
- Editor: vip
- Sudo: ssu
- Package Manager: bob
- File Manager: ccc
- Window Manager: dwl
- Dmenu: mew
- Misc: mandoc, libressl, zlib-ng
